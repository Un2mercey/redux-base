# Redux Base

## File of contents
<p>src/without-redux/index.js - application without redux.</p>

## Installation
1. [NodeJS](https://nodejs.org/)

```
node -v
v12.19.0

npm -v
6.14.8
```

2. Dependencies

```
npm i
```

## Starting
```
npm start
```
