# Redux Base
Basic redux usage.

## File of contents
src/without-redux/index.js - application without redux.
src/native-redux/index.js - application with custom redux.
src/index.js - application with [redux](https://redux.js.org/).

## Installation
1. [NodeJS](https://nodejs.org/)

```
node -v
v12.19.0

npm -v
6.14.8
```

2. Dependencies

```
npm i
```

## Starting
```
npm start
```
